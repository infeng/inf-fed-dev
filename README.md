# inf-fed-dev

frontend dev boilerplates

## Getting Started

```bash
# Install
$ npm install inf-fed-dev -g

# Create app
$ inf react my-app

# Start app
$ cd my-app
$ npm start
```

## Commands

we have 1 commands: `react`.

### inf react <appName>

create a simple react app.

#### Usage Examples

```bash
inf react my-app
```
